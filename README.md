# Data Structures and Algorithms (DSA) with Java

Master DSA fundamentals with Java. This course covers core data structures—arrays, linked lists, trees, and graphs—alongside essential algorithms like searching, sorting, and dynamic programming. Learn to optimize code, analyze time complexity, and build strong problem-solving skills to crack top-tier technical interviews.

---

## 🚀 Features & Topics Covered

### 1. Data Structures
*   **Linear:** Arrays, Linked Lists (Singly, Doubly, Circular), Stacks, and Queues.
*   **Non-Linear:** Binary Trees, Binary Search Trees (BST), AVL Trees, Heaps, and Graphs.
*   **Advanced:** Hashing (HashMaps, HashSets) and Tries.

### 2. Algorithms
*   **Searching & Sorting:** Binary Search, Bubble, Insertion, Selection, Merge, and Quick Sort.
*   **Approaches:** Recursion, Backtracking, Greedy Algorithms, and Dynamic Programming (DP).
*   **Graph Algorithms:** BFS, DFS, Dijkstra’s, and Prim’s/Kruskal’s.

### 3. Complexity Analysis
*   Big O Notation ($O(1)$, $O(\log n)$, $O(n)$, $O(n \log n)$, $O(n^2)$)
*   Time and Space Complexity optimization.
