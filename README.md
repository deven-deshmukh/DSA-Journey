# 📂 Course Syllabus: Data Structures & Algorithms with Java

## 🗺️ Module 1: Basics & Logic Building
*   **Java Fundamentals Refresher:** Arrays, Loops, Conditionals, and Static Methods.
*   **Pattern Printing:** 
    *   *Basic:* Right-angled triangles, inverted pyramids, and basic squares.
    *   *Advanced:* Diamond shapes, Floyd's triangle, Pascal's triangle, and hollow blocks.
*   **Basic Math for DSA:** Prime numbers, GCD/LCM, Palindrome numbers, and Armstrong numbers.

---

## 🧩 Module 2: Essential LeetCode Patterns (Linear Data)
*   **Time & Space Complexity:** Introduction to Big O, Omega ($\Omega$), and Theta ($\Theta$) notations.
*   **Two Pointers Pattern:** Pair with Target Sum, Remove Duplicates, Squaring a Sorted Array.
*   **Sliding Window Pattern:** Maximum Sum Subarray of Size K, Longest Substring with K Distinct Characters.
*   **Fast & Slow Pointers (Tortoise & Hare):** Linked List Cycle Detection, Middle of the Linked List, Happy Number.
*   **Merge Intervals Pattern:** Merge Intervals, Insert Interval, Interval Intersections.

---

## 📈 Module 3: Searching & Sorting Paradigms
*   **Linear Search:** Searching in unsorted spaces.
*   **Binary Search Pattern:** 
    *   Order-Agnostic Binary Search, Ceiling/Floor of a Number.
    *   **Search in Modified Spaces:** Search in a Sorted Infinite Array, Search in Rotated Sorted Array.
*   **Sorting Algorithms:** Bubble Sort, Selection Sort, Insertion Sort.
*   **Divide & Conquer Sorting:** Merge Sort and Quick Sort.
*   **Cyclic Sort Pattern:** Find the Missing Number, Find all Duplicates in an Array (Crucial for array-index manipulation).

---

## 🔗 Module 4: Core Linear Data Structures
*   **Arrays & ArrayLists:** Resizable array implementations, matrix manipulation.
*   **Linked Lists:** Singly, Doubly, and Circular Linked Lists; In-place Reversal of a Linked List pattern.
*   **Stacks & Queues:** Implementation via arrays/lists, Min-Stack implementation.
*   **Monotonic Stack/Queue Pattern:** Next Greater Element, Daily Temperatures, Sliding Window Maximum.

---

## 🌲 Module 5: Hierarchical Structures & Tree Patterns
*   **Binary Trees & BSTs:** Properties, Structural Validations, and Traversals.
*   **Tree Breadth-First Search (BFS) Pattern:** Binary Tree Level Order Traversal, Zigzag Traversal.
*   **Tree Depth-First Search (DFS) Pattern:** Path Sum, All Paths for a Sum, Sum of Path Numbers.
*   **Heaps / Priority Queues:** Min-Heap and Max-Heap implementations.
*   **Top 'K' Elements Pattern:** Kth Largest Element in a Stream, Top 'K' Frequent Numbers.
*   **K-way Merge Pattern:** Merge K Sorted Lists, Kth Smallest Number in M Sorted Lists.

---

## 🔄 Module 6: Advanced Problem-Solving & Backtracking
*   **Recursion:** Subsets pattern (Generating Subsets, Permutations, String Traversals).
*   **Backtracking:** N-Queens problem, Sudoku solver, and Combinational Sum.
*   **Bit Manipulation Pattern:** Single Number, Flip Image, Two Single Numbers.

---

## 🕸️ Module 7: Advanced Non-Linear Structures & Graph Patterns
*   **Hashing:** Internal working of `HashMap` and `HashSet` in Java, collision handling.
*   **Island (Matrix DFS/BFS) Pattern:** Number of Islands, Flood Fill, Max Area of Island.
*   **Graphs:** Representation, BFS, DFS, Dijkstra's Shortest Path, and Bellman-Ford algorithms.
*   **Topological Sort (Graph Pattern):** Task Scheduling, Course Schedule I & II, Alien Dictionary.
*   **Tries:** Longest Common Prefix, Word Search II.

---

## 💎 Module 8: Optimization Paradigms
*   **Greedy Algorithms:** Fractional Knapsack, Huffman Coding, and Job Scheduling.
*   **Dynamic Programming (DP) Patterns:**
    *   **0/1 Knapsack:** Subset Sum, Equal Subset Sum Partition.
    *   **Unbounded Knapsack:** Coin Change, Rod Cutting.
    *   **Fibonacci Numbers:** Climbing Stairs, House Robber.
    *   **Longest Common Subsequence (LCS):** Edit Distance, Longest Palindromic Subsequence.
